A Timestamp Microservices. A basic project using Node.JS, Express, Postman and JavaScript.

The project uses GET method to fetch the params, which are checked via conditions inside get method to be a perfect UNIX or time stamp and return a JSON object with properties unix value and UTC time for the entered params. Clone and run the project at localhost:3000/api/timestamp/:date_string. "date_string" is the user entered params here.
Perfect for basic understanding of how backend services are made.
